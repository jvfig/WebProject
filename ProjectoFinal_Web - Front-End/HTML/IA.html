Segundo a Google é um conjunto de tecnologias que permite que os computadores desempenhem uma variedade de funções avançadas, tais como a capacidade de ver, perceber e traduzir :

  • linguagens escritas e faladas
  • analizar dados
  • fazer recomendações
  • e muitas outras.

Tal como existe uma inteligência humana de origem biológica, a inteligência artificial seria em tudo semelhante à primeira, excepto na sua origem.

  • IA Limitada ou Fraca
    Corresponde ao único tipo de IA que existe actualmente. Só executa tarefas para as quais foi treinada, de forma muito mais célere e eficaz que qualquer humano. No entanto, não executa tarefas para além daquelas que recebeu treino.
  • IA Geral ou Forte
    Consegue executar tarefas para além daquelas que recebeu treino. A partir das suas aprendizagens e capacidades passadas consegue realizar novas tarefas num novo contexto sem a necessidade de ser treinada para o efeito.
  • Super IA
    Corresponderia literalmente a uma inteligência humana : seria capaz de pensar, raciocinar, aprender, de fazer julgamentos, de perceber as emoções humanas e de ter ela própria sentimentos, crenças e desejos.

  • Máquinas Reactivas
    Sâo sistemas de IA sem memória que foram projectadas para desempenhar uma tarefa específica. Como não conseguem memorizar dados ou decisões passadas, elas trabalham apenas com os dados disponíveis no momento. Elas reagem aos dados que recebem sem haver uma aprendizagem. Exemplo : IBM Deep Blue.
  • Memória Limitada
    Sâo sistemas de IA com memória em que podem utilizar dados e/ou previsões passados ou presentes para tomarem decisões. Exemplos : Generative AI, Virtual Assistants and Chatbots, Self-driving Cars.
  • Teoria da Mente
    Sâo sistemas de IA que conseguiriam perceber os pensamentos e emoções de outras entidades. Isto permitiria simular o comportamento humano. Também conseguiriam perceber e contextualizar a arte humana e os artigos de opinião, que a "Generative AI" é actualmente incapaz de o fazer.
  • Auto-consciência
    Sâo sistemas de IA que seriam capazes de fazer introspeções e teriam as suas próprias emoções, necessidades e crenças.

IA Multimodal
Tipo de IA que consegue perceber e gerar múltiplas formas de dados simultaneamente, tais como :
  • texto
  • imagens
  • sons

É o caminho para chegar à IA. Corresponde a uma subcategoria de IA que utiliza algoritmos de aprendizagem automática para :

  • reconhecer padrões a partir de dados
  • aplicar o conhecimento obtido dos padrões para tomar cada vez mais melhores decisões.

Aprendizagem Supervisionada
Tipo de aprendizagem na qual os algoritmos aprendem a partir de dados catalogados. O algoritmo recebe os dados e as correspondentes etiquetas. O objectivo deste tipo de aprendizagem é prever eficazmente as etiquetas para novos dados, os quais são desconhecidos para o algoritmo.
Aprendizagem Não Supervisionada
Tipo de aprendizagem na qual os algoritmos aprendem a partir de dados não catalogados. O objectivo deste tipo de aprendizagem é descobrir padrões, relações ou estruturas.
Aprendizagem Semi-supervisionada
Tipo de aprendizagem híbrida que combina a aprendizagem "Supervisionada" e "Não Supervisionada". O objectivo deste tipo de aprendizagem é superar a limitação associada aos dados catalogados utilizando os dados não catalogados para obter informação adicional e um contexto. Esta abordagem é utilizada para evitar os custos elevados ou o consumo de tempo associados à obtenção de dados catalogados.
Aprendizagem por Reforços
Tipo de aprendizagem inspirada no comportamento humano de aprendizagem por tentativa e erro. Através de um sistema de recompensas e penalidades, o algoritmo vai ajustando as suas decisões.

São algoritmos de aprendizagem profunda (Deep Learning Algorithms que constituem um dos sub-ramos de "Machine Learning") que conseguem :

  • reconhecer
  • resumir
  • traduzir
  • prever
  • recomendar
  • e gerar conteúdo utilizando grandes conjuntos de dados.

É o processo a partir do qual os sistemas de IA "Generative AI" são guiados para criar os conteúdos desejados.

Prompt
É a linguagem natural escrita (texto) que é utilizada para fazer pedidos aos sistemas de IA "Generative AI".
Generative AI
É um sistema de IA que cria conteúdos novos, tais como :
  • histórias
  • conversas
  • vídeos
  • imagens
  • música
  • etc.

Sâo ambos sistemas de IA "Generative AI" que utilizam LLM.

ChatGPT foi desenvolvido em parceria pela OpenAI e pela Microsoft e foi lançado em 2022. GPT significa "Generative Pre-Trained Transformer" :

  • "Generative" deriva dos sistemas de IA "Generative AI"
  • "Pre-Trained" advém do facto de ter sido preparada com padrões da linguagem humana com o intuito de produzir conteúdo escrito que faça sentido e que seja o mais verdadeiro possível
  • "Transformer" é o algoritmo que permite processar mega dados e condensá-los num texto simples, coerente como se de uma conversa se tratasse.

Bard foi desenvolvido pela Google e foi lançado em 2023. A escolha do nome foi provavelmente inspirada no homónimo "bard" que significa poeta ou contador de histórias.
