Estamos numa era extremamente excitante :

  • a inteligência arficial (IA) já está presente no nosso quotidiano, como por exemplo na condução autónoma...
  • está-se na iminência de descobrir um material ou uma liga de materiais que permita que a computação quântica funcione à temperatura ambiente...
  • o turismo espacial está a desenvolver-se...
  • há planos para criar uma base permanente na Lua...
  • há planos para levar o ser-humano a Marte...

...sejam bem-vindos ao meu website...

Tendo em conta os critérios de avaliação :

  • todo o website foi feito utilizando as tecnologias obrigatórias - HTML5, CSS3 e Bootstrap ;

  • para trazer dinamismo e lógica à página utilizou-se JavaScript (tecnologia não mencionada e apreendida na sua maior parte em modo auto-didacta) ;

  • era pedido um " website" no mínimo com 2 webpages com um formulário de contacto - utilizaram-se 3 "webpages" + 1 "webpage" para o formulário de contacto ;

  • não me considero uma pessoa particularmente criativa, mas uma pessoa que chega a uma solução possível e que depois a trabalha para a tornar numa melhor solução, ou seja vejo-me como alguém pragmático ;

  • a estilização cromática do "website" foi baseada em 5 tons : claros, de castanho, azuis, cinzentos e escuros ;

  • as opções cromáticas foram baseadas nos meus gostos pessoais, ou seja, elas são subjectivas ;

  • todo o "website" foi pensado para ser interactivo : a maioria dos elementos presentes no "website" senão mesmo todos podem ser alvo de interação com os utilizadores ;

  • a interacção do "website" com os utilizadores é importante na medida em que permite mostrar aos utilizadores que o "website" não é passivo (ou seja, responde aos estímulos deles e isso permite captar a atenção deles) e além disso funciona como uma medida de orientação (pelo elemento que está a interagir ficam a saber em que ponto da "webpage" estão situados) ;

  • todo o "website" foi projectado para ser dinâmico para quebrar a monotonia associada à estática (elementos que estão sempre imóveis, que nunca mudam de estado...);

  • os temas escolhidos para o "website" (inteligência artificial, computação quântica e UFO / UAP) reflectem as minhas preferências pessoais podendo estar ou não interligados entre si ;

  • todo o "website" foi orientado para ser "responsive" com o auxílio da "framework" Bootstrap ;

  • tendo em conta os pontos supracitados, o efeito final conseguido com o meu "website", todo o meu esforço dedicado a este projecto e a todas as aulas e a minha evolução durante este período eu acho que mereço um !
